## 🚢 项目快速熟悉指南

### 📋 项目概述
这是一个**港口堆场规划优化系统**，主要目标是在保证港口运营效率的前提下，通过智能算法优化集装箱的堆场分配，减少拥堵并节约成本。

### 🏗️ 核心架构
项目采用模块化设计，包含4个主要模块：

1. **数据预处理模块** (`preprocessing/`)
2. **预测模型模块** (`predict/`)
3. **仿真流程模块** (`simulation/`)
4. **结果分析模块** (`eda/`)

#### 2. 核心文件定位
- **主入口**: `simulation/allocate/docker_final.py` - 仿真和分配策略
- **分配算法**: `allocation_policy()` 函数（第2000行左右）
- **数据预处理**: `preprocessing/simu_processing/data_preprocessing_v0.py`
- **配置数据**: `data/blocks_config.json` - 堆场物理配置

#### 3. 关键概念理解

**堆场区域划分**：
- 空箱区：`*0X`, `*MA`, `*MB` 等
- 危险品区：`*1E`, `*1C`, `*8B`
- 冷冻箱区：`*2F`, `*6R`, `*6P`
- 超限箱区：`*4R`, `*5R`, `*4P`

**集装箱属性**：
- 尺寸：20尺、40尺
- 类型：空箱(E)、重箱(F)
- 特殊属性：冷藏、危险品、超限

#### 4. 运行流程
1. **准备数据**: 将原始数据文件放入 `data/` 目录
2. **预处理**: 运行 `node_path_calculator.py` 和 `data_preprocessing_v0.py`
3. **预测**: 运行两个预测模型的训练和预测流程
4. **仿真**: 执行 `docker_final.py` 进行堆场分配仿真
5. **分析**: 使用 `eda.py` 分析结果

#### 5. 调试建议
- 从调试模式开始：`--is_debug 1`
- 查看 `result.txt` 了解最新运行结果
- 使用前端可视化：`simulation/frontend/`

#### 6. 技术栈
- **后端**: Python + Flask + Pandas + NumPy
- **前端**: Vue.js (需要Node.js环境)
- **机器学习**: PyTorch + Scikit-learn
- **数据处理**: Pandas + NumPy
